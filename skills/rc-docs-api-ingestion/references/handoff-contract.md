# Handoff Contract: RC Docs & API Ingestion

## Required Fields

- from: rc-docs-api-ingestion
- to: downstream skill
- objective: one-sentence target
- artifact_paths: changed files or published links
- decisions: key choices and rationale
- open_risks: unresolved concerns
- validation_state: checks performed and status

## Acceptance Check

Handoff is accepted when artifacts are usable without re-discovery and next actions are explicit.

## Template

from: rc-docs-api-ingestion
to: next-skill
objective: concise objective
artifact_paths:
  - path/or/url
decisions:
  - decision with rationale
open_risks:
  - unresolved risk
validation_state:
  passed:
    - check
  failed:
    - check
