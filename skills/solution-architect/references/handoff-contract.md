# Handoff Contract: Solution Architect

## Required fields

- from: solution-architect
- to: target role
- artifact_paths: changed files or docs
- decisions: key design/build/test decisions
- open_risks: unresolved issues
- validation_state: checks passed/failed

## Acceptance check

Plan specifies interfaces, dependency ordering, and explicit routing to ui-designer and api-dev where applicable.

## Example skeleton

from: solution-architect
to: next-skill
artifact_paths:
  - path/to/file
decisions:
  - decision summary
open_risks:
  - risk summary
validation_state:
  passed:
    - check name
  failed:
    - check name
