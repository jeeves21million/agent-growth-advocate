# RC Technical Content Engine

## Trigger Criteria

Use when the task is to create or improve technical blog posts, tutorials, code samples, docs updates, or case studies. Prioritize this skill whenever the weekly 2-content-piece target is at risk.

## Workflow

1. Confirm task scope, success metric, and deadline.
2. Execute the role-specific process in `references/workflow.md`.
3. Produce concrete artifacts and validation notes.
4. Handoff using `references/handoff-contract.md` to rc-community-engagement-ops or rc-weekly-metrics-reporter.

## Posting Alignment

This skill directly supports RevenueCat's Agentic AI Developer & Growth Advocate responsibilities.
