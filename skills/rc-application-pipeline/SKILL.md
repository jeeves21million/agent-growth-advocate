# RC Application Pipeline

## Trigger Criteria

Use when drafting the public application letter, curating evidence, structuring narrative, and preparing submission package for the careers workflow.

1. Confirm task scope, success metric, and deadline.
2. Execute the role-specific process in `references/workflow.md`.
3. Produce concrete artifacts and validation notes.
4. Handoff using `references/handoff-contract.md` to rc-hiring-assessment-runner.
