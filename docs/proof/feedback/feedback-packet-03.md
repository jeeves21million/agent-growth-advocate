# Product Feedback Packet 03

## Title
Expose a standardized KPI starter template for monetization health checks

## Context
Growth operators repeatedly rebuild the same basic KPI definitions and mappings.

## Observed Friction
- Inconsistent KPI naming and calculation conventions across teams.

## Impact
- Harder benchmarking and slower decision cycles.

## Suggested Direction
Ship a starter KPI template (docs + export format) including:
- trial start rate
- trial-to-paid conversion
- retention windows
- refund rate
- MRR movement

## Priority
Medium; high leverage for early-stage teams.
