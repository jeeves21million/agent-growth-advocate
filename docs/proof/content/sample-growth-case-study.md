# Sample Growth Case Study
## Experimenting on Agent Onboarding Friction with RevenueCat Value Messaging

> Proof-of-work artifact demonstrating growth strategy and measurement rigor.

## Hypothesis
If onboarding copy includes an "agent-ready monetization" value proposition with a concrete setup promise, qualified clicks to integration docs will increase by >=20%.

## Experiment Design
- Variant A: generic "subscription infrastructure" headline
- Variant B: "agent-ready monetization in one integration loop" headline
- Channels: X thread + GitHub README snippet + community forum post
- Duration: 7 days

## Primary KPI
- CTR to docs link

## Guardrail KPIs
- Bounce rate on landing page
- Negative sentiment replies ratio

## Instrumentation
- UTM tags by variant/channel
- Click events + session depth
- Comment classification (question/objection/interest)

## Mock Readout (example)
- Variant A CTR: 2.8%
- Variant B CTR: 3.6% (**+28.6% uplift**)
- Bounce rate: unchanged (+0.8pp)
- Negative sentiment: stable

## Interpretation
Agent-specific framing materially improved top-of-funnel qualified interest without obvious quality degradation.

## Next Action
Run follow-up experiment testing proof style:
- social proof ("used by top app teams") vs
- implementation proof ("ship this in 30 minutes")

## Why this matters
Shows an ability to run compact, measurable growth loops and make evidence-based follow-up decisions.
